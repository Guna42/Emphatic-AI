## рҹ«Җ What is Empathica AI?

**Empathica AI** is a fine-tuned language model built to do what most AI systems fail at вҖ” *actually understand how you feel.*

Built on Google's `Flan-T5` architecture and supercharged with **LoRA (Low-Rank Adaptation)** fine-tuning, Empathica generates responses that are warm, emotionally aware, and genuinely supportive. Not robotic. Not cold. *Human.*

It also ships with a **side-by-side comparison mode** вҖ” so you can see exactly how fine-tuning transforms a generic base model into something that *feels* different.

## рҹ§¬ The AI Behind It

### Why Flan-T5?
Google's `flan-t5-base` is an instruction-tuned transformer that's compact, fast, and strong at following natural language instructions вҖ” making it ideal for empathetic response generation without needing massive compute.

### Why LoRA / PEFT?
**Low-Rank Adaptation** means we don't retrain the entire model (billions of parameters). Instead, we inject small trainable matrices into the base model вҖ” making fine-tuning:
- рҹҡҖ Faster to train
- рҹ’ҫ Cheaper to store (just the adapter, not the whole model)
- рҹҺҜ Highly targeted to empathetic responses

```
Base Model (frozen)  +  LoRA Adapter (fine-tuned)  =  Empathica AI вңЁ
```

### Inference Flow
```
User Input  в”Җв”Җв–¶  Prompt Engineering  в”Җв”Җв–¶  Tokenizer  в”Җв”Җв–¶  Flan-T5 + LoRA
                                                                  в”Ӯ
                                                                  в–ј
User sees warm, empathetic reply  в—Җв”Җв”Җ  Detokenize  в—Җв”Җв”Җ  model.generate()
```

## рҹҡҖ Getting Started

### Prerequisites
- Python 3.10+
- pip
- (Optional) Docker

---

### рҹ–ҘпёҸ Local Setup

**1. Clone the repo**
```bash
git clone https://github.com/Guna42/Emphatic-AI.git
cd Emphatic-AI
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Start the backend**
```bash
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

**4. Open the frontend**

Just open `frontend/index.html` in your browser (or use VS Code Live Server).

> вң… That's it. Talk to Empathica.

## рҹ”Ң API Reference

### `POST /generate`

Generate an empathetic response.

**Request:**
```json
{
  "text": "I've been feeling really overwhelmed lately and don't know what to do."
```

**Response:**
```json

  "reply": "I'm really sorry you're feeling this way. It's completely okay to feel overwhelmed sometimes вҖ” it means you care deeply. Let's take it one step at a time together."
```

## рҹ”¬ Comparison Mode

One of Empathica's coolest features вҖ” open `compare.html` to see **Base Flan-T5 vs Empathica AI** side by side.

| Input | Base Flan-T5 | Empathica AI |
