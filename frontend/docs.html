Overview

Empathica AI provides a simple REST API built with FastAPI. The API runs locally on port 8000.

BASE URL: http://localhost:8000

GET /health

Returns system status and the device the models are running on. Use this to confirm the server started and models loaded correctly.

Response

JSON Response · 200 OK 
{
  "status": "running",
  "device": "cpu",
  "model": "Empathica AI v1.0"
}
FieldTypeDescription
statusstring"running" when healthy
devicestring"cpu" or "cuda"
modelstringModel version identifier

POST /generate

The core endpoint. Accepts a customer complaint and tone preference, runs it through the two-model pipeline (DistilRoBERTa emotion detection + Flan-T5 with LoRA fine-tuning), and returns an empathetic response.

Request Body

Content-Type: application/json 
{
  "complaint": "My order hasn't arrived in 3 weeks and no one is helping me!",
  "tone": "friendly"
}
FieldTypeDescription
complaintstringRequired. The customer's complaint text.
tonestringOptional. friendly, formal, or apologetic. Defaults to friendly.

Response

JSON Response · 200 OK 
{
  "emotion": "anger",
  "response": "Thank you for contacting Amazon Support. We sincerely apologize for the inconvenience...",
  "tone": "friendly"
}
FieldTypeDescription
emotionstringDetected emotion (e.g. anger, sadness, neutral)
responsestringThe AI-generated empathetic support response
tonestringEcho of the tone used to generate the response

Code Examples

Easily interact with the API using Python or standard fetch requests.

Python · requests 
import requests

url = "http://localhost:8000/generate"

payload = {
    "complaint": "My order hasn't arrived in 3 weeks and no one is helping me!",
    "tone": "friendly"
}

response = requests.post(url, json=payload)
data = response.json()

print(f"Detected Emotion: {data['emotion']}")
print(f"Generated Response: {data['response']}")
